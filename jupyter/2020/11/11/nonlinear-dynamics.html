import numpy as np 
import matplotlib.pyplot as plt

How does the climate change over time? In the last lecture we saw that we our simple model is already quite good simulating how the climate changes over time.

Our simple model is based on ordinary differential equations (ODEs), where some variables change in timee - the rate of change as function of their current values.

$$ \frac{dx(t)}{dt} = f(x(t)) $$

We already implemented the simplest numerical method to solve such an equations: (forward) Euler method.

$$ \frac{dx(t)}{dt} = \frac{x(t+\Delta t) - x(t)}{\Delta t}$$

with the approximation

$$ x(t+\Delta t) = x(t) + \Delta t f(x(t)) $$

Solving the ODE: Euler method

Let's try to solve the equation

$$ \frac{dx}{dt} = x(1-x) $$

using the Euler method:

$$ \frac{x(t+\Delta t) - x(t)}{\Delta t} = x (1-x) $$ $$ x(t+\Delta t) = x(t_0) + \Delta t*(x(t_0))$$

Using the notation from the previous lecture we summarize:

$$ x_{n+1} = x_n + \Delta t \cdot \text{tendency}(x_n; ...)$$ 

def ode(x, timesteps):
    x = np.asarray(x)
    for i in np.diff(timesteps):
        if x.size == 1:
            x = np.append(x, x + i * x * (1-x))
        else:
            x = np.append(x, x[-1] + i * x[-1] * (1-x[-1]))
    return x

t = np.arange(0,20, 0.01)
dxdt = ode(0.5, t)

plt.plot(t, dxdt)
